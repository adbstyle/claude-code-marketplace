# Claude Code Marketplace — Adrian Bader

Plugin marketplace for Claude Code skills by [@adbstyle](https://github.com/adbstyle).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **requirementsengineer** | Analysiert und dokumentiert Anforderungen als User Stories und Akzeptanzkriterien nach IREB-Standards |

## Installation

```bash
# 1. Marketplace hinzufügen
/plugin marketplace add adbstyle/claude-code-marketplace

# 2. Verfügbare Plugins ansehen
/plugin

# 3. Plugin installieren
/plugin install requirementsengineer@adbstyle-skills
```

## Auto-Updates

After installing, enable auto-updates for this marketplace:

1. Run `/plugin` in Claude Code
2. Go to **Marketplaces** tab
3. Select **adbstyle-skills**
4. Enable **Auto-update**

When a new version is pushed, your plugins will update automatically.
